MS-CIT म्हणजे काय?

Maharashtra State Certificate in Information Technology – महाराष्ट्र शासनाने मान्यता दिलेला संगणक शिक्षण कोर्स

📌 MS-CIT म्हणजे काय?

MS-CIT (Maharashtra State Certificate in Information Technology) हा महाराष्ट्र शासन मान्यताप्राप्त संगणक कोर्स आहे. MKCL (Maharashtra Knowledge Corporation Limited) द्वारे हा कोर्स तयार केला गेला आहे.

हा कोर्स 120 तासांचा असतो आणि संगणकाचे मूलभूत ज्ञान, Microsoft Office, Internet Skills आणि Digital Literacy यावर लक्ष केंद्रित करतो.

महाराष्ट्रातील सरकारी नोकरींसाठी MS-CIT अत्यावश्यक आहे. शाळा, महाविद्यालये, बँका, सरकारी विभाग – सर्वत्र MS-CIT आवश्यक आहे.

📚 अभ्यासक्रम

MS-CIT मध्ये काय शिकवले जाते?

संपूर्ण MS-CIT अभ्यासक्रम – प्रत्येक विषय सविस्तर माहितीसह

💻

Module 1: Computer Fundamentals

संगणकाची ओळख
  • Computer म्हणजे काय – परिचय व इतिहास
  • Hardware आणि Software मधील फरक
  • Input Devices – Keyboard, Mouse, Scanner
  • Output Devices – Monitor, Printer, Speaker
  • Storage Devices – HDD, SSD, Pendrive

संगणकाची सुरुवात 1940 पासून झाली. आज संगणक हा आपल्या जीवनाचा अविभाज्य भाग बनला आहे. Hardware म्हणजे संगणकाचे दिसणारे भाग – जसे Monitor, Keyboard, Mouse. Software म्हणजे संगणकातील कार्यक्रम – जसे Windows, MS Office. या Module मध्ये तुम्ही संगणकाची संपूर्ण रचना समजून घ्याल.

🪟

Module 2: Operating System

Windows वापर
  • Windows Desktop Interface ओळख
  • File आणि Folder तयार करणे, व्यवस्था करणे
  • Control Panel – System Settings
  • Taskbar, Start Menu वापर
  • Recycle Bin, Shortcuts, Properties

Windows हे सर्वात जास्त वापरले जाणारे Operating System आहे. या Module मध्ये तुम्ही Desktop वापरणे, Files Organize करणे, System Settings बदलणे, आणि Windows च्या सर्व मूलभूत गोष्टी शिकाल. हे ज्ञान नोकरी आणि घरी दोन्हीकडे उपयुक्त आहे.

⌨️

Module 3: Typing Skills

Keyboard & Speed
  • Keyboard Layout – सर्व Keys ची ओळख
  • Home Row Typing Technique
  • English आणि Marathi Typing
  • Typing Speed वाढवण्याचे तंत्र
  • Accuracy आणि Speed Practice

Typing हे संगणक वापराचे सर्वात महत्त्वाचे कौशल्य आहे. वेगवान आणि अचूक Typing केल्यामुळे Office काम, Data Entry, आणि Exam मध्ये फायदा होतो. आमच्याकडे नियमित Typing Practice Sessions असतात.

🌐

Module 4: Internet Skills

Digital World
  • Internet म्हणजे काय – कसे काम करते
  • Browser वापर – Chrome, Firefox
  • Search Techniques – Google Search
  • File Download आणि Upload करणे
  • Online Services, YouTube, News

Internet हे आधुनिक जगाचे माहिती केंद्र आहे. या Module मध्ये तुम्ही Browser वापरणे, Google वर Search करणे, Online Forms भरणे, Files Download करणे आणि Internet सुरक्षितपणे वापरणे शिकाल.

📧

Module 5: Email Communication

Professional Communication
  • Gmail Account तयार करणे
  • Email पाठवणे आणि प्राप्त करणे
  • Attachments – Files पाठवणे
  • Professional Email लिहिणे
  • Inbox Manage करणे, Spam

Email हे व्यावसायिक संवादाचे मुख्य साधन आहे. Job Application, Office Communication, Documents Share करण्यासाठी Email अत्यावश्यक आहे. आमच्याकडे Email लिहिण्याचे Professional Training दिले जाते.

Most Important
📝

Module 6: Microsoft Word

Document Creation
  • Document तयार करणे, Save करणे
  • Text Formatting – Font, Size, Bold, Italic
  • Tables तयार करणे आणि Manage करणे
  • Page Setup, Margins, Headers, Footers
  • Resume / Biodata तयार करणे
  • Document Print करणे

Microsoft Word हे सर्वात जास्त वापरले जाणारे Document Software आहे. Letters, Reports, Applications, Resumes, Biodata – सर्व Word मध्ये तयार होतात. Office कामात, शाळेत, College मध्ये, सरकारी कार्यालयात सर्वत्र Word वापरले जाते. आमच्याकडे Professional Resume बनवण्याचे Training दिले जाते.

Career Booster
📊

Module 7: Microsoft Excel

Data & Calculations
  • Data Entry – Rows, Columns, Cells
  • Formulas – SUM, AVERAGE, COUNT, IF
  • Charts आणि Graphs तयार करणे
  • Data Sort, Filter करणे
  • Business Records, Accounts Manage करणे

Microsoft Excel हे Data Management साठी अत्यावश्यक Tool आहे. Bank, Office, Business, School – सर्वत्र Excel वापरले जाते. Salary Sheets, Attendance Records, Stock Management – सर्व Excel मध्ये होतात. Excel शिकल्यावर Data Entry, Accountant, Office Assistant या नोकऱ्या मिळण्यास मदत होते.

📑

Module 8: Microsoft PowerPoint

Presentation Skills
  • Presentation तयार करणे
  • Slide Design – Themes, Layouts
  • Animations आणि Transitions
  • Images, Charts Insert करणे
  • Slideshow Present करणे

PowerPoint हे Presentations साठी वापरले जाणारे Software आहे. Seminars, Meetings, Project Presentations, Class Presentations – सर्वत्र PowerPoint वापरले जाते. या Module मध्ये तुम्ही Professional Presentations तयार करायला शिकाल.

📱

Module 9: Digital Skills

Modern Digital Life
  • Online Forms भरणे – Scholarship, Aadhar
  • Government Services – DigiLocker, Seva
  • Digital Payments – UPI, Net Banking
  • Online Banking वापर
  • Internet Safety आणि Cyber Security

Digital India च्या युगात Online Services वापरता येणे अत्यावश्यक आहे. Online Scholarship Applications, Ration Card, Aadhar Services, Income Tax Filing – सर्व Online होते. या Module मध्ये तुम्ही सर्व Digital Services वापरायला शिकाल.
